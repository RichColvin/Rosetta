Rosetta
Software Library 		 

These files are used with the Rosetta manual, Instructions for Building.

To save any of the files, click on the file's name on the right side of the screen (e.g., "7i92t_rosettad.bin").

Step 1 - Get software & other files needed to setup Rosetta

Step 1a - Mesa card support software

Version Files
1.0
  1. 7i92t_rosettad.bin

  2. 7i92t_rosettad.pin

  3. PIN_RosettaD_34.vhd

Download Instructions

Download these files to the Downloads library on the Raspberry Pi.

/home/cnc/Downloads

Final Location

These files get moved later into the library,

/usr/lib/firmware/hm2/hostmot2

Usage

These files get used for configuring the system and the Mesa card.
 

Step 1b - Rosetta applicaton files

Version Files
1.0
  1. Rosetta.hal

  2. Rosetta.ini
⭐ These files will be updated once we get the final configuration completed. There may be more. The goal is to get the configuration so that we only need to place files onto the system and not use pncconf for each machine. If nothing else, these will make the process repeatable.

Download Instructions

Download these files to the Downloads library on the Raspberry Pi.

/home/cnc/Downloads

Final Location

These files get moved later into the library,

/home/cnc/Desktop/Rosetta

Usage

These files define the system and Mesa card connectons.
