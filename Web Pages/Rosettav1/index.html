Overview

The Rosetta system is designed to

  1. provide the ornamental turner a way for linking activities using synchronisation of stepper motors, and
  2. assist the ornamantal turner with automating onerous tasks.

It is based on LinuxCNC running on a Raspberry Pi. Details about the configuration are in the Rosetta library.

Axis Colours

The colours used for the axes are the ones designated for that respective axis. The use of colour coding for each axis is to help the ornamental turner ensure they are pluging the stepper motor cable into the correct receptacle for that axis. The cabling and stepper motor enclosures should match those colours.
